AWS Shell in minimal footprint
============================

Pass environment variables for 

```
AWS_ACCESS_KEY_ID
AWS_SECRET_ACCESS_KEY
AWS_DEFAULT_REGION
```

Get quick and easy access to aws-shell in a Docker container
